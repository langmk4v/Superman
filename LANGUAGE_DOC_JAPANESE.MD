## 変数
`var` キーワードを使用して、変数を定義します。<br>
```
var num = 123;
```
コロン `:` で型を明示的に指定することも出来ます。
また、シャドウイング可能です。
```
var str : string = "aiueo";
var str = 20; // shadowing
```

## 関数
### 定義
`fn` で関数を定義します。<br>
引数リストの後に `-> T` と書くと戻り値の型を指定できます。(省略可能) <br>
```
fn func() {
}

fn func2(
```
オーバーロードはできません。<br>
```
fn func() {
    ...
}

fn func(n: int) { // ERROR! Duplicated name.
```

また、定義する前に呼び出すことが出来ます。<br>
Fire 言語では C, C++ のような前方宣言は不要で、変数以外の全ては定義する前に参照可能です。<br>
```
println(get_hello());

fn get_hello() { return "Hello"; }
```

### テンプレート
Fire では関数テンプレートを使用できます。
```
fn add <T> (a: T, b: T) -> T {
    return a + b;
}

println(add(1, 2));          // => 3
println(add("abc", "def"));  // => abcdef
```
